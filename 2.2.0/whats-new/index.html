    What's on This Page

    The big stuff. The shiny new things. The stuff we got tired of fixing. For everything else, there's the changelog.

    Chunklet v2.2.0

    ‚ú® Simpler Chunking API

    We renamed some methods. Yes, we're those people who rename things. But honestly, the old names were confusing ‚Äî even to us:

    • chunk_text() ‚Äî chunk a string
    • chunk_file() ‚Äî chunk a file directly
    • chunk_texts() ‚Äî batch strings
    • chunk_files() ‚Äî batch files

    The old chunk and batch_chunk still work. They'll whine at you with a deprecation warning. Deal with it or migrate ‚Äî your choice.

    üîó PlainTextChunker Got Absorbed

    PlainTextChunker is now part of DocumentChunker. We know ‚Äî having two chunkers was weird. Just use chunk_text() or chunk_texts() like a normal person. The old import still works, technically, with a deprecation warning.

    ‚úÇÔ∏è SentenceSplitter Now Does split_text()

    split() is out. split_text() is in. We renamed it because apparently "split" was too short. There's also now split_file() if you're the type who likes skipping steps.

    üé® Visualizer Makeover

    The chunk visualizer finally got some love:

    • Fullscreen mode ‚Äî for when you want to pretend you're doing something important
    • 3-row layout ‚Äî less cluttered, more clickable
    • Smoother hovers ‚Äî no more seizure-inducing animations
    • Smarter buttons ‚Äî they stay enabled because, honestly, disabling them was stupid

    ‚å®Ô∏è Shorter CLI Flags

    Finally, stuff you can actually type without wrist strain:

    • -l for --lang
    • -h for --host
    • -m for --metadata

    You're welcome.

    üßë‚Äçüíª Code Chunking, Less Broken

    Code chunking got slightly less terrible:

    • Cleaner output ‚Äî fixed weird artifacts in chunks from comment handling (we know, it was annoying)
    • More languages ‚Äî Forth, PHP 8 attributes, VB.NET, ColdFusion, and Pascal. Yes, really.
    • String protection ‚Äî multi-line strings and triple-quotes won't get mangled anymore

    üîß The Boring But Necessary Stuff

    • Tokenizer timeout ‚Äî new --tokenizer-timeout / -t flag so custom tokenizers don't hang forever
    • Direct imports ‚Äî from chunklet import DocumentChunker now works without making things slow
    • Fewer crashes ‚Äî fixed dependency issues with setuptools<81 in CI (sentsplit and pkg_resources, long story)
    • Global registries ‚Äî custom_splitter_registry and custom_processor_registry exist now
    • Error messages ‚Äî slightly less cryptic when things explode

    Chunklet v2.1.1

    üêõ Visualizer Was Broken

    The visualizer didn't work after installing from PyPI. Static files were MIA. Fixed now, obviously.

    Chunklet v2.1.0

    üåê Visualizer 1.0

    We built an actual UI. Because sometimes you want to click buttons instead of writing code:

    • Interactive web interface for parameter tuning
    • Launch with chunklet visualize
    • Works with all chunker types

    üìÅ More File Formats

    ODT, CSV, and Excel (.xlsx) ‚Äî added in this release. Because apparently plain text wasn't enough for some people.
